# FullStack Flask + React + Mysql

## Backend Flask

1. Set up virtual env
2. select interpreter
3. install dependency
4. Create file app.py
5. run flask

```bash
# run 
flask run
```

## FrontEnd React

1. install dependency
2. run vite

```bash
# run 
npm run dev
```

## Database Mysql

1. configure new database
2. import data from backend/database/seed.sql
